Plugin Pack #3

[ GENERATE SKY ]

The [ GENERATE SKY ] Puzzle creates a 'Sky' object.
The Sky object provides a shader to reproduce a Sky environment compete with a sun.

The settings 'inclination' and 'azimuth' designate the position in the sky the sun will be given.
Users can also select to use the 'Sky' object's 'sun position' to add a sun 'Light' object using the [ GENERATE SUN ] plugin puzzle.
.

[ GENERATE SUN ]

The settings 'inclination' and 'azimuth' designate the position in the sky the sun will be given.
Users can also select to use the sun position in the sky created by the [ GENERATE SKY ] plugin puzzl using the designated checkbox.
.

This is a convenient method of describing a three dimensional space when designing or displaying a 3D scene.
This grid helper expands in all directions infinitely.
