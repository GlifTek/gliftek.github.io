Plugin Pack #1

[ LENS FLARE DELUXE ]

The [ LENS FLARE DELUXE ] Puzzle creates a Lens Flare Object with three optional effects:
'Rays', 'Glow', and 'Bokeh',
each of the three Effects are found within their own corresponding 'Settings' puzzle with a healthy set of customizable options (shown further below).




The 'Settings' puzzles can be used singularly, or stacked into lists and then applied via variable puzzles.
Users can stack as many settings as desired to create dynamic 'Rays' effects.

If inputs of any Effects Setting are left blank, default settings will be used.

Purchasers of the Lens Flare Deluxe Plugin can check the pack's accompanied Verge3D project application files for examples like shown.

[ RAYS SETTINGS ]

The [ RAYS SETTINGS ] Puzzle allows for control of multiple spreads of symmetrical or non-symmetrical rays of light from the center of the Lens Flare Object.

Users can determine symmetry, ray count, ray width, ray sharpness, randomness of ray length, rotation, blur, opacity, size, distance, and color.

The following options are explained:

Symmetry:
This designates whether the rays will be positioned symmetrically. If the 'ray count' option has an odd number chosen while symmetrical option is checked, the number of rays will be rounded up to the nearest integer.

Ray Count:
This decides how many rays are produced.

Ray Width:
This option changes the width of the rays.

Ray Sharpness:
Determines the sharpness of the ray.

Randomness Of Ray Length:
This option randomizes the rays lengths. A number between 1 and 10 is allowed. 0 negates this.

Rotation:
Rotates the entire group of rays.

Blur:
Sets the level of blur of the rays.

Opacity:
The opacity of the rays

Size:
The overall size of the entire group of rays

Distance:
The distance from the light to the camera. A value of 0 (nearest to light) to 1 (nearest to camera) is used. This option is convenient for ordering any stacked settings one may have.

Example:
Set a stack of of Rays puzzles setting their 'distance' between .01 to .05, a large background Glow at 0, and a strong small center Glow at .06, at the very front.

Color:
designates the color of the rays.

[ GLOW SETTINGS ]

The [ GLOW SETTINGS ] Puzzle controls a single or multiple radial gradients of tone starting from the center of the Lens Flare Object.



Users can designate gradient tone spacings via three control fields.
Rotation, blur, and opacity are also available.

[ BOKEH SETTINGS ]

The [ BOKEH SETTINGS ] Puzzle creates bokeh style fragmented in-lens effects. A number of shapes are available to chose from.

Of such are: line, circle, triangle, square, pentagon, hexagon, heptagon, octagon, & nonagon polygon shapes.

Users can determine ray width, ray length, and randomness of ray length.
